# Frontend

## Stack Frontend

- Nextjs!!!!!!
- Typescript

### Specs

- O frontend deve ter um bom UX
- O frontend deve ser simples, vamos fazer apenas um MVP
- TODAS AS REQUISIÇÕES DEVEM SER PASSADAS E TRATADAS PELO BACKEND _ANTES!!!!!!!!!!!!_ DE IR PARA O BANCO DE DADOS
- O frontend não deve aceitar sqlinjection
- Perfomance é algo muito legal de se ter no frontend
- Deve ter tela de Loading para quando o usuário estiver esperando uma resposta
- Não deve demorar mais do que 1 segundo para carregar a página
- Se for necessário carregue a parte da página que não precise de dados e coloque um loading onde precisa enquanto não estiver pronto
- O design deve ser totalmente responsivo
- Esta liberado usar bibliotecas ou libs para fazer o UI
